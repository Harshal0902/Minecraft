## Prerequisites

Required to install and run the software:

 * [npm](https://www.npmjs.com/get-npm)


## Installing and Running

From the project folder, run these commands in console (terminal) to install dependencies and run the app:
```
1. npm install
2. npm run dev
3. npm start
```
